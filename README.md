# Task3 с^(-1) mod m=d

Обратный элемент числа c по модулю m существует только тогда, когда `gcd(c,m)=1`. Для его нахождения используется расширенный алгоритм Евклида: `если коэффициенты тождества удовлетворяют равенству cu + mv = 1, то число u mod m является обратным к c по модулю m`

# Task4 Hughes 

Алгоритма позволяет Alice генерировать ключ и послать его Вob:
(1) Алиса выбирает случайное большое целое число х и генерирует
`k=gx  mod n`
(2) B выбирает случайное большое целое число у и посылает А
`Y=gy  mod n`
(3) А посылает B
`Х=Yx mod n`
(4) B вычисляет
`z=y-1 mod( п-1)`

`k’=Xz  mod n`
Если все выполнено правильно, k=k'.

# Task6 

Представление числа в виде цепной дроби строится последовательным делением с остатком, при котором частные записываются в виде элементов цепной дроби. Для решения линейного уравнения `Ъ × а + Ь × b = D` используется расширенный алгоритм Евклида: `целые решения существуют тогда и только тогда, когда D делится на gcd(A,B)`, после чего по найденному частному решению записывается общее решение уравнения.
